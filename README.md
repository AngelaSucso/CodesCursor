# Cursor con Python: Desarrollo Inteligente con IA

## ¿Qué es Cursor?

Cursor es un entorno de desarrollo integrado (IDE) basado en Visual Studio Code que incorpora inteligencia artificial directamente en el flujo de programación. A diferencia de un editor tradicional, Cursor permite interactuar con la IA sobre el código, recibir sugerencias contextuales, generar funciones completas y refactorizar archivos teniendo en cuenta el proyecto como un todo.

## Acerca del Repositorio

Este repositorio contiene una colección de prácticas y ejemplos desarrollados en el marco del curso de Santander Open Academy. Los ejercicios fueron implementados utilizando Python y el entorno Cursor, aprovechando sus capacidades de autocompletado inteligente, chat integrado y asistencia en la escritura y comprensión del código.

## Objetivo

El objetivo de este repositorio es aprender Python utilizando herramientas de IA como apoyo en el desarrollo. A través de ejemplos prácticos, se explora cómo Cursor puede ayudar en tareas como:
- comprensión de código existente,
- generación de scripts desde cero,
- corrección de errores,
- refactorización y documentación.

Los ejercicios incluyen distintos tipos de proyectos, como:
- scripts de consola,
- análisis de datos,
- automatización de tareas,
- aplicaciones web simples,
- interfaces gráficas de escritorio.

## Estructura del Repositorio

- **prueba**: Carpeta utilizada para experimentación inicial con Cursor y pruebas rápidas de código.
- **introduccion**: Ejercicios básicos para familiarizarse con Python y con el uso de Cursor como asistente de programación.
- **ejercicios_simples**: Ejemplos prácticos de algoritmos y lógica básica, desarrollados con apoyo del autocompletado y las sugerencias de la IA.
- **ejemplos**: Proyectos más completos que integran múltiples conceptos, como análisis de ventas, gestores de tareas y scripts de automatización.

Cada subcarpeta incluye su propio archivo README.md con instrucciones y explicaciones específicas.

## Cómo Ejecutar los Ejemplos

Los scripts están escritos en Python (versión 3.7 o superior). Se recomienda el uso de un entorno virtual para evitar conflictos de dependencias durante la ejecución. Los archivos pueden ejecutarse directamente desde la terminal o utilizando las opciones de ejecución integradas en el entorno Cursor.

## Rol de Cursor IA en el Desarrollo

Cursor IA fue utilizado como apoyo durante el desarrollo de los ejercicios, facilitando la escritura de código, la explicación de fragmentos complejos y la corrección de errores. Su integración directa con el editor permitió iterar rápidamente sobre las soluciones y comprender mejor cada paso del proceso, convirtiéndose en una herramienta clave para el aprendizaje práctico de Python.
