# Proyecto Prueba

## Descripción

Este es un proyecto de ejemplo en Python que contiene una función simple para imprimir números del 0 al 5. El proyecto demuestra la estructura básica de un script de Python con una función y un punto de entrada principal.

## Requisitos

- Python 3.6 o superior

No se requieren dependencias externas adicionales ya que el proyecto utiliza solo la biblioteca estándar de Python.

## Instalación

1. Clona o descarga este repositorio en tu máquina local.

2. Asegúrate de tener Python instalado. Puedes verificar la versión ejecutando:
```bash
python --version
```

3. No es necesario instalar dependencias adicionales ya que este proyecto solo utiliza la biblioteca estándar de Python.

## Uso

Para ejecutar el script, utiliza el siguiente comando desde la línea de comandos:

```bash
python ejemplo.py
```

Esto ejecutará la función `imprimirNumeros()` que imprimirá los números del 0 al 5 en la consola.

### Salida esperada

```
0
1
2
3
4
5
```

## Estructura del Proyecto

```
prueba/
├── ejemplo.py       # Script principal con la función imprimirNumeros()
└── README.MD        # Este archivo
```

## Funcionalidades

- **imprimirNumeros()**: Función que itera sobre un rango de números del 0 al 5 e imprime cada número en una nueva línea.
