# Introducción

Esta carpeta contiene ejercicios introductorios del curso Cursor con Python. Los archivos son ejemplos independientes que demuestran diferentes conceptos básicos de programación en Python.

## Archivos

### ejercicio_autocompletar.py

**Descripción:**  
Este archivo contiene un juego interactivo donde el usuario debe adivinar un número secreto entre 1 y 100. El programa genera un número aleatorio y proporciona pistas indicando si el número ingresado es mayor o menor que el número secreto. El juego continúa hasta que el usuario adivine correctamente o cancele el juego.

**Conceptos que se practican:**
- Generación de números aleatorios con `random.randint()`
- Bucles `while` para controlar el flujo del juego
- Estructuras condicionales (`if`, `elif`, `else`)
- Manejo de excepciones (`try`, `except`) para validar entrada del usuario
- Validación de entrada de datos
- Interacción con el usuario mediante `input()` y `print()`
- Uso de `break` para salir de bucles

**Cómo ejecutar:**
```bash
python ejercicio_autocompletar.py
```

---

### prueba.py

**Descripción:**  
Este archivo contiene varios ejemplos básicos de programación en Python que incluyen: un saludo inicial, un bucle que imprime números, y un algoritmo para calcular y mostrar números primos menores que 100. Es un archivo de práctica que demuestra conceptos fundamentales de manera simple.

**Conceptos que se practican:**
- Función `print()` para mostrar mensajes
- Bucles `for` con `range()`
- Anidación de bucles (bucles dentro de bucles)
- Estructuras condicionales
- Algoritmos matemáticos (cálculo de números primos)
- Operadores aritméticos y módulo (`%`)
- Manejo de listas (creación y manipulación)
- Operador `break` para salir de bucles anidados

**Cómo ejecutar:**
```bash
python prueba.py
```
